Derzost

Derzost is a functional one-page website for an original board game «Something with Something» designed to foster creativity. The site serves as a platform where users can contribute to game development, learn about the rules and features, and place pre-orders.

The project was developed using HTML, CSS, and JavaScript, ensuring a smooth and interactive user experience. PHP is employed for handling all feedback forms. The responsive design of the site guarantees optimal viewing across all devices, from mobile phones to desktop computers.


Website Type:

Landing page with feedback forms.

Technologies:

HTML, CSS, JavaScript, PHP
