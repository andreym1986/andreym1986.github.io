Safepoint

Safepoint is a landing page designed for an autonomous floor-standing device that enables contactless hand disinfection. This device is ideal for public spaces where safe and touch-free hand sanitization is essential.

The project was developed using HTML, CSS, and JavaScript, ensuring a clean and modern approach to coding. The form for submitting applications is implemented using PHP. The site's responsive design ensures it is easily accessible on various devices, from phones to desktop computers.


Website Type:

Landing page with a contact form submission feature.

Technologies:

HTML, CSS, JavaScript, PHP
